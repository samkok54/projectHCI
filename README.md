Overview
========
 Movie Rating Application
 แอพให้เรตคะแนนภาพยนตร์

Features
========

- ดูรายละเอียดของภาพยนตร์
- เรียงลำดับเงินจากมากไปน้อย ,น้อยไปมาก


Use cases
=========
- เพิ่มภาพยนตร์
- เพิ่มรายละเอียดภาพยนตร์
- ดูรายละเอียดภาพยนตร์
- ให้คะแนนภาพยนตร์
- ลบภาพยนตร์
- คอมเม้นต์เกี่ยวกับภาพยนต์เรื่องนี้
